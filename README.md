# Spec Kit Plus

This project implements a comprehensive textbook on Physical AI & Humanoid Robotics using Docusaurus framework. The project features:

- 6 structured chapters following the 4 course modules
- RAG chatbot for AI-powered learning assistance
- Better-Auth integration for user authentication and personalization
- Urdu translation capabilities
- Deployment-ready for GitHub Pages or Vercel

## Development Setup

1. Navigate to the `my-website` directory
2. Install dependencies: `npm install`
3. Start development server: `npm run start`

## Auto-commit Feature

This project includes custom Claude Code commands for automated git operations:
- `/git:push` - Automatically stages, commits, and pushes changes
- `/git:safe-push` - Same as above but with additional safety checks
- `/git:auto-push` - Commits with Claude-generated commit messages
